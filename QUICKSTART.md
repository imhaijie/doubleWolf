# 🚀 快速开始指南

## 5 分钟上手狼人杀面杀助手

### 第一步：启动服务器

```bash
# 进入项目目录
cd werewolf-helper

# 启动开发服务器
npm run dev

# 或者生产环境
npm start
```

服务器将在 `http://localhost:3000` 启动。

### 第二步：创建房间

1. 打开浏览器访问 `http://localhost:3000`
2. 点击"创建房间"
3. 输入你的名字
4. 选择游戏设置：
   - 发言时间：10 分钟（推荐）
   - 投票时间：120 秒（推荐）
5. 选择角色配置：9 人标准局（推荐）
6. 点击"创建房间"

### 第三步：分享房间码

- 房间码显示在页面顶部（例如：`ABC123`）
- 分享给朋友："来玩狼人杀！房间码：ABC123"
- 朋友访问 `http://localhost:3000` 输入房间码加入

### 第四步：选择身份

1. 所有玩家进入房间后
2. 每人选择自己的双身份：
   - 第一身份：选择一个角色
   - 第二身份：选择另一个角色
3. 等待所有人都选好

### 第五步：开始游戏

1. 房主点击"开始游戏"按钮
2. 进入第 1 夜

### 第六步：夜晚行动

根据系统提示进行操作：

**狼人**：
- 看到"选择击杀目标"
- 点击要击杀的玩家
- 点击"确认击杀"

**预言家**：
- 看到"选择查验目标"
- 点击要查验的玩家
- 点击"查验"
- 查看结果（查杀/金水）

**女巫**：
- 看到"解药"界面
- 选择救/不救被刀玩家
- 进入"毒药"界面
- 选择毒杀一名玩家或不用

**守卫**：
- 看到"选择守护目标"
- 点击要守护的玩家
- 点击"确认守护"

**平民**：
- 看到"等待中..."
- 等待其他玩家行动

### 第七步：白天阶段

**死亡公告**：
- 系统显示"【X 号】第一身份阵亡"
- 3 秒后自动消失

**发言阶段**：
- 顶部倒计时开始（10 分钟）
- 玩家依次发言
- 白狼王可以点击"自爆"按钮

**投票阶段**：
- 倒计时结束自动进入
- 选择要投票出局的玩家
- 点击"投票给 X 号"
- 等待其他玩家投票

**猎人开枪**（如触发）：
- 猎人被投票出局
- 选择要带走的玩家
- 点击"开枪"

### 第八步：游戏结束

当满足胜利条件时：
- 系统显示获胜阵营
- 公开所有玩家身份
- 查看操作时间轴

## 📱 手机访问

### 电脑作为服务器

1. 查看电脑 IP 地址：
   - Windows: `ipconfig`
   - Mac/Linux: `ifconfig`
   - 找到类似 `192.168.x.x` 的地址

2. 手机访问：
   - 打开浏览器
   - 输入 `http://电脑 IP:3000`
   - 例如：`http://192.168.1.100:3000`

### 确保同一 WiFi

- 电脑和手机必须连接同一 WiFi
- 某些公共 WiFi 可能限制设备间通信

## 🎯 常见问题

### Q: 无法加入房间？
A: 检查：
- 房间码是否正确（6 位，区分大小写）
- 房间是否已满
- 游戏是否已开始

### Q: 断线了怎么办？
A: 不用担心：
- 重新访问房间链接
- 系统自动恢复你的身份和状态
- 游戏继续进行

### Q: 房主掉线了？
A: 
- 房主 24 小时内重连可恢复
- 超过 24 小时房间自动清理
- 可以重新创建房间

### Q: 提示音不响？
A: 
- 首次需要点击页面任意位置
- 浏览器限制自动播放音频
- 检查设备音量

### Q: 角色颜色看不清？
A: 
- 系统使用深色主题
- 调整屏幕亮度
- 在明亮环境下使用

## 💡 新手建议

### 第一次玩双身份？

1. **记住你的两个身份**
   - 第一身份死亡后第二身份生效
   - 狼人身份死亡后无法使用狼人技能

2. **注意阵营归属**
   - 任一身份是狼人 = 狼人阵营（永久）
   - 好人要找出所有狼人

3. **胜利条件不同**
   - 好人：淘汰所有狼人阵营
   - 狼人：淘汰所有金水宝宝或神职

### 角色优先级

**好人阵营**：
1. 预言家：尽早跳身份报查验
2. 女巫：合理使用解药和毒药
3. 守卫：优先守护预言家
4. 猎人：死亡时带走狼人
5. 平民：积极发言投票

**狼人阵营**：
1. 狼人：夜晚刀人，白天伪装
2. 白狼王：找时机自爆带走神职

## 🎮 游戏变体

### 9 人标准局（推荐新手）
- 3 狼（含白狼王）+ 4 神 + 2 民

### 12 人标准局
- 4 狼（含白狼王）+ 4 神 + 4 民

### 自定义配置
- 房主可自由调整角色数量
- 建议保持狼人：好人 ≈ 1:3
